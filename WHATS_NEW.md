# 🎉 Что нового в версии 3.1

## ✨ Автоматическая загрузка групп

Теперь бот **автоматически сканирует Google Drive** и загружает все доступные группы!

### Было раньше:
- ❌ Нужно было вручную выбирать направление
- ❌ Нужно было угадывать букву группы
- ❌ Нужно было знать точный код

### Стало сейчас:
- ✅ **Бот сам находит все группы** на Drive
- ✅ **Группировка по направлениям** (ACC, AUD, BAN, BAT и т.д.)
- ✅ **Удобный список** всех групп
- ✅ **Пагинация** - по 15 групп на страницу
- ✅ **Навигация** - кнопки "Вперед/Назад"

## 📚 Как это работает

### Шаг 1: Форма обучения
```
🏫 Очное
🌙 Вечернее
📮 Заочное
🎓 Магистратура
```

### Шаг 2: Курс
```
📖 1 курс (1-LEVEL)
📖 2 курс (2-LEVEL)
📖 3 курс (3-LEVEL)
📖 4 курс (4-LEVEL)
📖 5 курс (5-LEVEL)
```

### Шаг 3: Направление
Бот автоматически сканирует Drive и показывает:
```
📚 ACC - Бухгалтерский учет (2 группы)
📚 AUD - Аудит (6 групп)
📚 BAN - Банковское дело (2 группы)
📚 BAT - Бизнес администрирование (5 групп)
...
```

### Шаг 4: Группа
Показывает все группы направления с пагинацией:
```
Страница 1 из 2:
📋 ACC-71U    📋 ACC-75R
📋 AUD-71U    📋 AUD-72U
...

[« Назад] [📄 1/2] [Вперед »]
```

### Шаг 5: Формат
```
📷 Фото (удобнее)
📄 PDF
```

## 🎯 Преимущества

1. **Не нужно знать коды** - просто выбираете из списка
2. **Видите все группы** - не угадываете есть ли ваша
3. **Быстрый поиск** - группировка по направлениям
4. **Удобная навигация** - пагинация для больших списков
5. **Всегда актуально** - данные берутся с Drive в реальном времени

## ⚙️ Настройка

Если структура вашего Drive отличается, просто отредактируйте `folder_structure.py`:

```python
# Главная папка
MAIN_FOLDER_ID = "ваш_id"

# Формы обучения
EDUCATION_TYPES = {
    "daytime": {
        "name": "🏫 Очное",
        "subfolder_name": "1. Кундузги таълим..."
    },
    ...
}

# Курсы
COURSES = {
    "1": "1-LEVEL",
    "2": "2-LEVEL",
    ...
}
```

Бот автоматически адаптируется!
